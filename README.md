# Bybit Trading Bot with Sentiment Analysis

An advanced cryptocurrency trading bot for Bybit that uses sentiment analysis and news monitoring to make informed trading decisions with strict risk management.

## Features

- **Sentiment Analysis**: Real-time market sentiment from news and social media
- **Risk Management**: Maximum 5% loss per trade protection
- **Automated Trading**: Fully automated execution on Bybit
- **News Integration**: Multiple news sources for market intelligence
- **Real-time Monitoring**: Continuous market analysis and position management

## Installation

1. **Clone the repository**
2. **Create virtual environment**
3. **Install dependencies**: `pip install -r requirements.txt`
4. **Setup environment variables**
5. **Run the bot**: `python trading_bot.py`

## Important Warning

⚠️ **Use at your own risk!**
- Always test with small amounts first
- Use Bybit testnet for development
- Monitor the bot regularly
